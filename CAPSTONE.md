# Florassence 🌿

## Capstone Reflection

Florassence was developed as my frontend internship capstone project. What started as a 4-day course assignment became a complete React Native application through continuous improvements and experimentation.

The main idea behind the project was to create a study companion that combines productivity tools with a more engaging experience.

## My Approach

During development, I focused on building the application in small steps:

- Started with the core UI and screen structure.
- Added navigation and different app flows.
- Connected external data through APIs.
- Added local storage for saving user data.
- Improved the user experience with validation, error handling, and offline support.

Each stage helped me understand how individual features come together to form a complete mobile application.


## Challenges & Solutions

### Managing Application State

Handling tasks, themes, authentication, and user preferences required proper organization.

**Solution:**  
Used React Context API and custom hooks to keep the code organized and make data easier to manage.

### Creating a Consistent Design

With multiple screens, maintaining the same look and feel was challenging.

**Solution:**  
Created a centralized theme system to manage colors, spacing, and styling across the application.

### Handling Real-World User Experience

A good app needs to handle situations like loading, errors, and no internet connection.

**Solution:**  
Added loading states, error messages, retry options, and offline indicators.


## Key Learnings

Through Florassence, I improved my understanding of:

- Building React Native applications from scratch.
- Structuring a scalable frontend project.
- Managing application state.
- Working with APIs and local storage.
- Creating better user experiences.
- Handling real device features and permissions.


## Final Thoughts

Florassence helped me move from building individual screens to understanding how complete applications are designed and developed.

The project strengthened my frontend development skills and gave me practical experience in turning an idea into a functional mobile application.
